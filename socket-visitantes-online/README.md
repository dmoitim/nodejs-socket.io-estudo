# socket-visitantes-online

Esse projeto simples retorna ao usuário a quantidade de usuários online na página.
Para executar o projeto siga os passos:
1- Clone o repositório;
2- Acesse o diretório do projeto;
3- Execute o comando npm install para instalar as dependências (nesse caso somente o socket.io);
4- Execute o comando node index.js, ou nodemon index.js caso você já possua o pacote nodemon instalado;
5- Acesso o endereço http://localhost:3000 em vários naveadores/abas e veja que o contador é incrementado/decrementado conforme você abre/fecha as abas.
