## socket-chat

Esse projeto cria um chat simples que permite a visualização das mensagens entre vários usuários.
Para executar o projeto siga os passos:

1- Clone o repositório;

2- Acesse o diretório do projeto;

3- Execute o comando npm install para instalar as dependências (nesse caso o express e o socket.io);

4- Execute o comando node index.js, ou nodemon index.js caso você já possua o pacote nodemon instalado;

5- Acesso o endereço http://localhost:3000 em vários naveadores/abas e ao enviar uma mensagem em um deles você verá que ela será atualizada em todos os demais.
